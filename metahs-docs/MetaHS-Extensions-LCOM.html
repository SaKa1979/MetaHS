MetaHS.Extensions.LCOM

Description

Metamodel Elements that are interesting (those that are used to determine nodes in MetaHS.EDSL.Graph.UsesGraph module ) are : TypeSynonym, DataType, Function. All other Elements that are part of the _contains relations are ignored.

LCOM4 - Number of connected components in the graph that represents each method as a node and the sharing of at least one attribute/method as an edge. ref. Hitz and Montazeri (1995)
