MetaHS.Extensions.LOC

Description

Metamodel Elements that are interesting (those that occupy distinct line numbers) are : ModuleHead, ModuleImport, TypeSynonym, DataType, Function, TypeSignature, TypeClass and Instance. All other Elements that are part of the _contains relations are ignored.

LOC - Sum of all non-empty, non-comment lines in a module.
